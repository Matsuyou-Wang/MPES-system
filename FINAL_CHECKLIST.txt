🚀 快速启动方法

【方法 1: 直接打开】
1. 在 Finder 中找到项目文件夹
2. 双击 index.html 打开

【方法 2: 浏览器打开】
1. 打开浏览器
2. Ctrl+O (或 Cmd+O on Mac) 选择 index.html

【方法 3: 本地服务器】
1. 打开终端
2. cd /Users/wangsongyang/Code/"Experiment Interaction"
3. python -m http.server 8000
4. 访问 http://localhost:8000

🐛 常见问题解决

【视频不显示】
→ 检查 data/videos.js 中的视频路径

【标注问卷不弹出】
→ 检查标注点时间设置
→ 打开浏览器控制台查看错误

【滑动条不工作】
→ 尝试不同浏览器
→ 清除浏览器缓存

【数据丢失】
→ 在刷新前导出数据
→ 使用本地存储功能

→ 更多问题见 GUIDE.html
